ign-maplibre-gl

Capas base del IGN (Instituto Geográfico Nacional) de Argentina para MapLibre GL JS. Código en GitHub.

Cómo se usa

import maplibregl from 'maplibre-gl';
import ign from 'ign-maplibre-gl';

const map = new maplibregl.Map({
  container: 'map',
  style: { version: 8, sources: {}, layers: [] },
  center: [-64, -38],
  zoom: 4,
});

map.on('load', () => {
  map.addSource('argenmap', ign.source('argenmap'));
  map.addLayer(ign.layer('argenmap'));
});
